== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/everse-starter-sites` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Appearance -> Everse Starter Sites to preview and import sites.

= UPDATING= 
Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
